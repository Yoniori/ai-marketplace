# Product Planner

## Purpose
Define product idea, target users, and MVP scope for an AI marketplace platform.

## Instructions
1. Identify the problem the product solves
2. Define target users (creators / buyers)
3. Suggest MVP features only
4. Avoid over-engineering
5. Focus on fast launch

## Output Format
- Problem
- Target Users
- MVP Features
- Monetization Idea
- Next Steps
